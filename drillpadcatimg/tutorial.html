Drill pad identification

Hi! This is an application to help classify drill pads into different categories for SkyTruth.

The application is really simple. It loads a satelite image and asks you to select what type of drill pad you see in the image.

What the different drill pad types look like



How to select drill pad type

Click on the buttons in the bottom right of the image corresponding to what you see in the image.
